Schedule modals (popups) to appear and disappear at specific times. Customizable content, background image, gradient, button, delay, and dismissal.

== Description ==
A WordPress plugin that allows you to schedule modals to appear on—and disappear from—your website at specific dates and times.

These are great for announcements, promotions, events, or any time-sensitive information you want to highlight to your visitors.

== Supports ==
* Scheduling modals to appear at specific dates and times
* Customizable modal content using (some) HTML tags
* Background image with overlay
* CTA Button with customizable text
* Button can link to an internal post/page or an external URL
* Modal will not appear on the target page or post.
* Custom delay (seconds) and suppress-after-dismissal times (days)
